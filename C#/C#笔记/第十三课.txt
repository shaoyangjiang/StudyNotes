多表连接：
1.内链接(inner join)
2.外连接：左外连接(left join)，右外连接(right join)
========================================================================
内链接语句：
第一种方法：
select ...
from <表一>
inner join <表二>
on ...
第二种方法：
select ...
from <表一>,<表二>
where ...
========================================================================
