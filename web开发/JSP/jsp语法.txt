1.JSP原始代码中包含了JSP元素和Template(模版)data两类。
==============================================================
2.Template data指的是JSP引擎不处理的部分，即标记<%.....%>以外的部分，例如代码中的HTML的内容，这些数据会直接传送到客户端的浏览器。
==============================================================
3.JSP元素则是指将由JSP引擎直接处理的部分，这一部分必须符合Java语法，否则会导致编译错误。
=============================================================
4.Template date 部分 
  <html>
  	<head>
  		<title>
  			Hi-Jsp实验<!-- Template data-->
  		</title>
  
  	</head>
  </html>
================================================================
5.JSP元素部分
  		<body>
  			<!-- JSP元素-->
  			<%
  			  String msg = "this is JSP test" ;
  			  out.println("Hello");
  			%>
  			<h1><% =msg %></h1>
  		</body>
==================================================================
6.JSP语法分为3种不同的类型
a.编译器指令(Directive) 例如：<%@ page import="java.io.*"%>
b.脚本语法(Scriping)
c.动作语法(Action) 例如：<jsp:forward>,<jsp:getProperty>,<jsp:include>
=======================================================================
7.注释
<!-- 这是HTML注释 -->
<%-- 这是JSP注释,隐藏注释，不会发送给客户端
 -->
====================================================================
8.声明
<%! int a = 3;>类似全局变量
===========================================================
9.表达式
<%= 表达式%>
==================================================================
10.转义符
双引号\",单引号\',反斜杠\\,回车\r,换行\n,制表符\t,退格\b.
=============================================================
11.包含指令
include指令：向当前页插入一个静态的文件的内容。
JSP语法格式：<%@ include file="相对位置"%> 
%>
============================================================
12.taglib指令
jsp语法<%@ taglib uri="URIToTagLibrary" prefix="tagPrefix" %>
用于引入定制标签库。
=============================================================
13<jsp:forward>
语法：<jsp:forward page={"relativeURL |""<%=expression%>"}/>
属性：page="{"relativeURL |""<%=expression%>"}“，这里是一个表达式或者是一个字符串用于说明你将要定向的文件或URL。这个文件可以是JSP或者程序段。
可以传参数：<jsp:param name="parameterName" value="{parameterValue |<%=expression %>}"/>
==================================================================
14<jsp:include>
语法：<jsp:include page={"relativeURL |""<%=expression%>" flush="true"}/>
可以传参数：<jsp:param name="parameterName" value="{parameterValue |<%=expression %>}"/>
