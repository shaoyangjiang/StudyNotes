连接断开服务器：
mysql -h localhost -u root -p [数据库名称(可选)]
密码sigma
mysql>quit
-------------------------------------------------------------------------------------
得到版本和当前日期：
mysql>select version(), current_date;
得到当前时间
mysql>select now();
-------------------------------------------------------------------------------------
将mysql用作一个简单的计算器：
//计算下面两个值
mysql>select sin(pi()/4), (4+1)*5;
-------------------------------------------------------------------------------------
以多行方式显示当前用户和日期：
mysql> select
    -> user()
    -> ,
    -> current_date;
注意：
- 会自动寻找终止分号来决定是否结束。
- 如果你决定不想执行正在输入过程中的一个命令，输入\c取消它：
-------------------------------------------------------------------------------------
显示所有数据库：
mysql>show databases;
-------------------------------------------------------------------------------------
创建并选择数据库：
mysql>create database testdatabase; //创建数据库
mysql>use testdatabase; //选择数据库，使他成为当前数据库
-------------------------------------------------------------------------------------
显示当前数据库中的所有表：
mysql>show tables;
-------------------------------------------------------------------------------------
创建数据库表的布局：
mysql> create table pet (name varchar(20),owner varchar(20),
    -> species varchar(20),sex char(1),birth date,death date);
-------------------------------------------------------------------------------------
显示pet表的列名称和类型等：
mysql>describe pet;
-------------------------------------------------------------------------------------
导入和插入数据：
导入：
先在txt中写入数据，一行一条记录，用制表符隔开，空的值用/N表示
然后把数据装载到表中：
/*******************************************
load data infile "pet.txt" into table pet ;//把文件放对路径
*********************************************/
mysql> LOAD DATA LOCAL INFILE '/path/pet.txt' INTO TABLE pet;
请注意如果用Windows中的编辑器（使用\r\n做为行的结束符）创建文件，应使用：
mysql> LOAD DATA LOCAL INFILE '/path/pet.txt' INTO TABLE pet
    -> LINES TERMINATED BY '\r\n';
（在运行OS X的Apple机上，应使用行结束符'\r'。）
插入：
mysql> INSERT INTO pet
    -> VALUES ('Puffball','Diane','hamster','f','1999-03-30',NULL);
-------------------------------------------------------------------------------------
查询一个表中所有的记录：
mysql> select * from pet;
-------------------------------------------------------------------------------------
更新一个表里的一条数据：
mysql> update pet set sex='m' where name = 'Dog';
-------------------------------------------------------------------------------------
搜索内容不重复：
select DISTINCT owner[列名] from pet;
-------------------------------------------------------------------------------------
显示当前所操作的哪个数据库：
select database();
-------------------------------------------------------------------------------------
删除数据表：
drop table 表名
-------------------------------------------------------------------------------------
